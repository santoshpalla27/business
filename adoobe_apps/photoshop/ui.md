# Opening a file

## 1) Normal Way (Most Common)

- Open Photoshop
- Go to File ‚Üí Open
- Select your file
- Click Open

Shortcut:

- Windows: Ctrl + O
- Mac: Cmd + O

## 3) Drag & Drop Method (Fastest)

- Drag image from your folder
- Drop it into Photoshop window

## 4) Opening RAW Files (Important)

Raw is unprocessed image file from camera.

If you open a RAW file, it will first open in Camera Raw.

Steps:

1. Make basic adjustments
2. Click Open (bottom right)
3. It opens inside Photoshop

## Professional Setup Advice

Before creating file, ask:

- Where will this be used?
- Screen or print?
- Horizontal or vertical?

Decide platform first ‚Üí then create document.

## Beginner Mistakes

- Designing in wrong aspect ratio
- Using CMYK for Instagram
- Using 300 PPI for web (unnecessary)
- Not planning layout size first

## Pro Workflow Habit

# Photoshop UI Walkthrough (Very Simple & Practical)

Think of Photoshop like a working table.

- Left = Tools
- Center = Your Image
- Right = Controls

## 1) Top Menu Bar

This is the control center.

Important menus:

- File ‚Üí Open, Save, Export
- Edit ‚Üí Undo, Transform
- Image ‚Üí Image size, adjustments
- Layer ‚Üí New layer, merge, masks
- Select ‚Üí Selections
- Filter ‚Üí Blur, sharpen, effects
- View ‚Üí Zoom, guides

You don't need all at once.
File, Edit, Layer are used most.

## 2) Toolbar (Left Side)

This is where your tools live.

Important tools for beginners:

- Move Tool ‚Üí Move objects
- Marquee Tool ‚Üí Make square/rectangle selection
- Lasso Tool ‚Üí Freehand selection
- Crop Tool ‚Üí Resize frame
- Brush Tool ‚Üí Paint
- Eraser ‚Üí Remove
- Text Tool (T) ‚Üí Add text
- Zoom Tool ‚Üí Zoom in/out

Tip: Press and hold on tool icons to see more options.

## 3) Canvas (Middle Area)

This is your working area.
Your image appears here.

You can:

- Zoom in (Ctrl + +)
- Zoom out (Ctrl + -)
- Hold Spacebar to move around

## 4) Layers Panel (Right Side ‚Äì Most Important)

This is the heart of Photoshop.
Each element sits on its own layer.

Think of layers like transparent sheets stacked on each other.

You can:

- Hide layer (eye icon)
- Change order (drag up/down)
- Add new layer
- Add mask
- Adjust opacity

If something looks wrong ‚Üí check layers first.

## 5) Adjustments Panel

Used to change:

- Brightness
- Contrast
- Color
- Hue
- Black & white

Best practice ‚Üí Use adjustment layers instead of editing image directly.

## 6) Properties Panel

When you select something (text, shape, mask),
its settings appear here.

For example:

- Font size
- Color
- Mask feather

## 7) Bottom of Layers Panel

Small icons:

- New layer
- Delete layer
- Add mask
- Add adjustment

You will use these daily.

## Simple Understanding

Photoshop =

- Tools (left)
- Image (center)
- Layers & Controls (right)

That's it.

## Beginner Mistakes

- Editing on background layer directly
- Not naming layers
- Ignoring layers panel
