# Photoshop Layers Panel

## 1. What a Layer Really Is

### Simple

Each layer holds one element ‚Äî image, text, shape, adjustment.

### Professional

A layer is an independent pixel or vector container that can be modified without affecting other elements in the composition.

***You can:***

- Move it
- Hide it
- Mask it
- Blend it
- Change opacity

## 2Ô∏è‚É£ Types of Layers (Very Important)

### 1. Background Layer

- Locked by default
- Usually original image
- Limited editing flexibility

üëâ Pro move: Double click ‚Üí convert to normal layer.

### 2. Normal Pixel Layer

- Standard image layer
- Fully editable
- Used for:
  - Retouching
  - Adding elements
  - Compositing

### 3. Adjustment Layer (Non-Destructive Editing)

Examples:

- Brightness/Contrast
- Levels
- Curves
- Hue/Saturation
- Color Balance

Why it matters

It edits without touching original pixels.

You can:

- Turn it off
- Edit later
- Add mask to affect only part

Industry standard:
Always use adjustment layers for color work.

### 4. Text Layer

- Editable text.
- Still vector-based until rasterized.

Pro rule:
Never rasterize text unless necessary.

### 5. Shape Layer

- Vector shape.
- Scales without losing quality.

Used in:

- Social media design
- Branding
- UI mockups

### 6. Smart Object (Professional Level)

A protected container.

Used for:

- Non-destructive transformations
- Mockups
- Reusable elements

Pro workflow:
Convert important layers into Smart Objects before heavy edits.

## 7Ô∏è‚É£ Layer Masks (Most Important Concept)

## 8Ô∏è‚É£ Clipping Mask

Very common in product editing.

## Real Example (Product Editing)

1. Convert to Smart Object ‚Üí Protect it
2. Add Curves Adjustment ‚Üí Improve lighting
3. Use Mask on Curves ‚Üí Brighten only shoe
4. Add another mask ‚Üí Blur background only

That is professional workflow.
