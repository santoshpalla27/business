# RIGHT PANEL — INFORMATION & CONTROL
## 1. Histogram

Shows exposure distribution.

Professional Tip:
Check for clipping before editing.

### Shortcut: J → Show clipping warning

## 2. Quick Develop

Basic exposure adjustments inside Library.

Use this when:
You want batch correction without entering Develop.

### Professional Trick:

Use for:
- Quick exposure fix
- Quick white balance correction

Avoid:
Heavy color grading here.

## 3. Keywording

Add searchable tags.

Example:
Bride, Groom, Wedding 2026, Indoor, Canon R5

Professional photographers depend heavily on keywords.

## 4. Metadata Panel

Shows:

- Camera
- ISO
- Aperture
- Lens
- Date
- Copyright

### Professional habit: Always add copyright metadata during import.
