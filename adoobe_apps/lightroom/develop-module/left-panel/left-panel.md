# LEFT PANEL (Workflow & Version Control)

***This side is about management of edits, not actual editing.***

## 1. Navigator

### What It Does

Small preview of image.

### Used For

- Quick zoom control
- Checking overall framing

### Pro Tip

Use Navigator to jump between zoom levels instead of scrolling.



## 2. Presets

### Simple Explanation

Saved editing recipes.

### Technical

A preset stores slider values (non-destructive).

### When To Use

- Batch editing weddings
- Instagram consistency
- Brand color style

### When NOT To Use

Do not rely on presets to fix bad exposure.


### Professional Workflow

- Correct exposure first
- Apply preset
- Fine-tune

### Shortcut

- Hover preset → preview effect instantly



## 3. Snapshots

### What It Does

Creates saved versions of current edit state.

### Use Case

Create:

- Natural Look
- Cinematic Look
- Black & White Version

Snapshots are inside the same photo.

## 4. History

### What It Does

Shows every editing step chronologically.

### Use

- Go back to any step.

### Professional Tip

If edit becomes messy → step back instead of randomly adjusting sliders.


## 5. Collections (Optional View)

### Quick access to grouped photos while editing.




### LEFT PANEL SUMMARY

- Presets = Speed
- Snapshots = Versions
- History = Undo timeline
- Navigator = View control
