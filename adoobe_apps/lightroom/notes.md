# Lightroom is NOT Photoshop.

## Lightroom is:

- A RAW processing engine
- A color grading system
- A photo management system
- A non-destructive editor

It is built for speed + consistency + batch editing.

# Lightroom Versions

1. Adobe Lightroom Classic :- Classic = Professional desktop version with full control.

### Technical Explanation
 
- Catalog-based file management
- Stores edits in .lrcat
- Advanced modules (Library, Develop, Print, Web)
- Better for large shoots (weddings, commercial, events)

### Used For:

- Wedding photographers
- Commercial photographers
- Bulk editing 1000+ images
- Studio workflows

2. Adobe Lightroom :- Cloud version. Clean, simple, auto-sync across devices.

### Technical Explanation

- Cloud storage based
- Simplified interface
- AI-based search

Auto sync across mobile/desktop

### Used For:

- Instagram creators
- Content creators
- Travel photographers
- Fast editing on the go

### Professional Advice (Brutally Honest)

If you want:

- Serious photography career → Learn Lightroom Classic first
- Casual content creation → Lightroom CC is fine
- Professional workflow → Use both strategically

### Core Lightroom Philosophy

1. Non-destructive editing :- You never damage the original file.

#### Real Example:
You edit a wedding photo heavily. 2 years later client wants natural look.
You reset. Original is untouched.



### The Lightroom Workflow System (Golden Order)

This is industry standard order:

1. Import
2. Cull (select best images)
3. Basic Exposure Correction
4. White Balance
5. Contrast & Presence
6. Tone Curve
7. Color (HSL)
8. Color Grading
9. Detail (Sharpening / Noise)
10. Local Adjustments
11. Crop & Straighten
12. Export

Never randomly move sliders.

Editing is systematic.
