
 # lightroom cc vs lightroom classic

 lightroom cc is a cloud based photo editing software 

 this is a cloud based app can be used on mobile or web or pc and everything is synced across all devices and backed up to the cloud 

 lightroom classic is a desktop based photo editing software
 this is a local desktop app and everything is stored on the local desktop and not backed up to the cloud and is complex to learn and powerfull tool


 ## both have similar ui and almost same features 

 - but use cc if you want to edit photos on the go and have a simple interface and dont want to worry about storage 

 - use classic if you want more features and more control over your photos and dont  mind the complex interface and storage issues

## 2. Core Architecture (Very Important)

### 4. Professional Editing Workflow (Step-by-Step)

Follow this order strictly:

#### Step 1 ‚Äì Profile Selection

- Adobe Color (default)

#### Step 2 ‚Äì White Balance

#### Step 3 ‚Äì Basic Exposure Corrections

#### Step 4 ‚Äì Presence

#### Step 5 ‚Äì Tone Curve (Advanced Contrast Control)

#### Step 6 ‚Äì HSL (Professional Color Control)

- Hue ‚Üí Shift color tone
- Saturation ‚Üí Intensity

- Darken blue ‚Üí richer sky
- Increase orange luminance ‚Üí better skin tones

#### Step 7 ‚Äì Color Grading (Modern Workflow)

- Shadows ‚Üí cool tones
- Highlights ‚Üí warm tones

#### Step 8 ‚Äì Detail Panel (Sharpening)

Sharpening settings (starting point):

- Amount: 40‚Äì60

- Luminance: 10‚Äì25 for normal ISO

#### Step 9 ‚Äì Lens Corrections

Enable:
- Remove Chromatic Aberration
- Enable Profile Corrections

Always turn this on.

#### Step 10 ‚Äì Effects

- Vignette (subtle)
- Grain (for cinematic look)


#### 8. Common Beginner Mistakes

- Over-saturation
- Too much clarity
- Over-sharpening
- Ignoring white balance
- Editing without histogram awareness
